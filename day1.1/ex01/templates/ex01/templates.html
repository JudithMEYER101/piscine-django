Ex01 : Moteur de template.

Django’s template engine separates presentation from Python code. Views pass data to templates using a context.

Blocks

Blocks define replaceable areas in a base template. Child templates override these blocks.

For loops

The template language can loop over values using for ... in.

If structures

Templates can conditionally display content with if statements.

Context variables

Variables passed by a view can be displayed in templates using double braces.
